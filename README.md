# Student Management System

## Objective

A console-based Java application for managing student records efficiently. The project demonstrates Object-Oriented Programming (OOP), Java Collections Framework, file handling, and user input validation.

## Features

* Add new student records
* Display all students in a formatted table
* Search students by ID
* Search students by name
* Update student details
* Delete student records
* Calculate class average grade
* Save and load data using file handling
* Input validation and exception handling
